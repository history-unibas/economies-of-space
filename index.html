Github repositories of the Economies of Space project

This page documents existing Github repositories that have been developed in the ‘Economies of Space’ project.

More information about the project: https://dg.philhist.unibas.ch/en/bereiche/mittelalter/forschung/oekonomien-des-raums/.

Repository Description
annotation-scripts-dhb2024 Script to annotate a collection of sentences recursively to generate nested annotations.
database Scripts for creating and updating the relational project database and for analysing the data stored in it.
metadata Scripts for processing metadata of the Historical Land Registry of the City of Basel.
pix-plot Scripts for analysing image clusters generated with PixPlot.
teaching-research-seminar-fs2024 Demo-Jupyter notebooks and processed data from the historical land register of the city of Basel for a research seminar.
transkribus Scripts for interacting with the Transkribus platform using their REST API.
